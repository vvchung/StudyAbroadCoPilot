# 🚀 留學 AI 學伴 (Study Abroad AI Companion)
> **讓台灣學生的夢想不再卡在 SOP！** 
> 
> 這是為了解決台灣學生在申請出國深造時，面對「留學文件」的高牆而誕生的 AI 守護者。我們不只是翻譯，我們是你的戰友。

---

## 🌏 願景：打破「留學文件」的階級高牆 (The Vision)
在台灣，許多優秀的學生擁有改變世界的潛力，卻往往因為無法負擔昂貴的留學代辦費用（動輒數萬甚至十數萬台幣），而在撰寫 SOP (Statement of Purpose) 與 CV 時感到無助。

我們拒絕讓「金錢」成為「夢想」的過濾器。
我們利用最強大的 AI 技術，將專業顧問的思維邏輯轉化為每個人都能使用的工具。因為**每一份夢想，都值得被世界看見。**

---

## ⚡ 核心架構：Gemini 雙模態共生 (The "Vibe" Architecture)
為了提供最精準的建議，我們採用了領先業界的 AI 架構：

- **🧠 智慧大腦：Gemini 3.1 Pro**
  擁有強大的推理能力，能深度理解學生的個人特質，並運用 **STAR 法則** (Situation, Task, Action, Result) 將平凡的經歷轉化為閃耀的亮點。
- **🛡️ 視覺守護：PDF Multimodal Context**
  支援直接上傳學校的申請指南或個人 CV PDF。AI 不再只是空談，而是能根據「真實文件內容」提供精確到點的修改建議。

---

## 🔥 殺手級功能 (Impact Features)

### 1. 📝 STAR 經歷潤飾引擎
別再說「我很努力」！AI 會引導你挖掘具體數據與事例，用 **Show, Don't Tell** 的原則，讓你的經歷在招生官眼中活起來。

### 2. 📎 PDF 全文深度理解
直接上傳學校的官方 PDF 指南。AI 會自動對齊學校要求，確保你的 SOP 每一段都精準擊中招生官的痛點。

### 3. 📋 一鍵複製，夢想加速
優化後的文字，一鍵即可複製到你的正式文件中。我們減少所有不必要的摩擦，讓你專注於內容的深度。

### 4. ✨ 視覺化呼吸引導 (Coming Soon)
申請季的焦慮我們懂。未來將整合視覺化冥想功能，在 AI 幫你改文件的同時，也照顧你的心理健康。

## 🛠️ 技術棧 (Tech Stack)
- **Frontend**: React 18 + Vite (極速開發體驗)
- **Styling**: Tailwind CSS (現代、優雅的 UI 設計)
- **AI Engine**: Google Gemini 3.1 Pro API
- **Animations**: Framer Motion (流暢的互動體驗)
- **Icons**: Lucide React
